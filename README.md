# ordering
 一个迷你的单位食堂订餐系统
这是我学习Python 以来做的第一个应用。
一个用Flask制作的网站，前端使用了bootstrap和WTF,后端采用了Python和MySql。通过WSGI部署在Apache服务器上，相应配置写在app.wsgi文件中。

* ### 2019-7-26: 主要订餐功能完成，只有系统管理中的增加人员和充值功能未做
